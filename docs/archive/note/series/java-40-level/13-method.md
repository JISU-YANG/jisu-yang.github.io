## JAVA 40 LEVEL
# Lv.13 Method

> 메서드는 기능을 의미하며, 프로그램의 시작과 끝이다.

### 메서드의 필수 요소
- 접근제한자
  - public
  - protected
  - default
  - private
- 리턴타입
- 메서드명
- Block

### 메서드의 옵션 요소
- static / non-static
  미리 메모리에 할당 여부
- 아규먼트 (Arguments)
  필요한 파라미터의 타입
- 파라미터 (Parameter)
  외부에서 전달받은 아규먼트를 블록 내부에서 사용하기 위한 변수
